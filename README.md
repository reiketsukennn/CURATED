# CURATED - Cafe & Resto Directory App

**CURATED** is a high-fidelity iOS application prototype developed to solve the challenge of discovering hidden gem cafes and restaurants. Built with a minimalist aesthetic, the app focuses on seamless user experience and clean visual hierarchy.

## 🚀 Features
* **Curated Discovery:** Hand-picked cafe and restaurant recommendations.
* **Apple Maps Integration:** View locations and navigate directly within the app.
* **User Authentication:** Secure onboarding and account management via Firebase.
* **Dynamic UI:** Interactive carousel cards and refined navigation flow.

## 🛠️ Tech Stack
* **Language:** Swift
* **Framework:** SwiftUI
* **Backend:** Firebase (Authentication & Database)
* **Architecture:** MVVM (Model-View-ViewModel)
* **Design Tool:** Figma

## 🎓 Learning Journey
This project was built as part of my preparation for the **Apple Developer Academy Indonesia**. Through this self-initiated project, I mastered:
* Architecting scalable iOS apps using SwiftUI.
* Managing backend integration with `GoogleService-Info.plist` and Swift Package Manager.
* Implementing advanced UI components like continuous rounded corners and monochrome themes.
